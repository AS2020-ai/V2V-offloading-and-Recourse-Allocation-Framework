EXPECTED FORMAT — SUMO FCD XML

<fcd-export>
  <timestep time="0.00">
    <vehicle id="veh0" x="312.4" y="187.2" speed="13.5" angle="90"/>
    <vehicle id="veh1" x="421.1" y="203.8" speed="8.2"  angle="270"/>
  </timestep>
  <timestep time="0.50">
    ...
  </timestep>
</fcd-export>
How to generate from SUMO:
1. Install SUMO: sudo apt install sumo or download from sumo.dlr.de
2. Run: sumo -c your.sumocfg --fcd-output trace.fcd.xml --step-length 0.5
3. Upload the resulting trace.fcd.xml here

Auto-detects column names for id, time, x, y, speed

AUTO-DETECTED COLUMN NAMES

ID columns:    id, vehicle_id, vehicleId, node, car
Time columns:  time, t, timestamp, timestep
X columns:     x, lon, longitude, pos_x, X
Y columns:     y, lat, latitude, pos_y, Y
Speed columns: speed, v, velocity, spd (optional)
First row must be a header. Delimiter: comma or semicolon. Missing speed → estimated from position delta.
